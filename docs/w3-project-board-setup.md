# W3 GitHub Project Board Setup Guide

## Quick Setup Steps

### 1. Create Project Board

1. Go to: https://github.com/kendychae/handcrafted-haven
2. Click "Projects" tab
3. Click "New project"
4. Choose "Board"
5. Name: "WDD430 W3 Team Collaboration Board"
6. Description: "Project board for Week 3 assignment - team collaboration and task management"
7. Visibility: Public
8. Click "Create project"

### 3. Create Columns

Create these columns in order:

1. **Backlog** - Tasks not yet started
2. **In Progress** - Currently being worked on
3. **Review** - Completed but needs team review
4. **Done** - Completed tasks
